In addition to modifying the php.ini file, you also need to add a Phalcon file to the Apache mods directory. Check the code comments in this code example: http://docs.phalcon.io/en/latest/reference/install.html#compilation . None of the comments are for MAMP, but you should be able to figure out where to put the file.
