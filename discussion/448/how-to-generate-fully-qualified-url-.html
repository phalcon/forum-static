I want to create a HTML mail letter.

How I can write a fully qualified url into html? I need to print something like

<a href="http://my.domain.com/some/action/param1/param2/">http://my.domain.com/some/action/param1/param2/</a>

I have a route:

$router->add('/some/action/{param1}/{param2}', 'something::somewhat', ['GET', 'POST'])->setHostName($domain)->setName('some-action');

This method does not allow to print hostname and print a label with same URL:

{% raw %}{{{% endraw %} link_to(['for': 'some-action', 'param1': '123', 'param2': '456'], "my link") {% raw %}}}{% endraw %}

And this method does not allow to 'compile' route with my parameters and does not allow to create link with custom url:

{% raw %}{%{% endraw %} set some_route = routes['password-new'] {% raw %}%}{% endraw %}
{% raw %}{%{% endraw %} set some_url = domain ~ some_route.compilePattern(['param1': '123', 'param2': '456']) {% raw %}%}{% endraw %}
{% raw %}{{{% endraw %} link_to(some_url, some_url) {% raw %}}}{% endraw %}

Where "domain" is a "http://my.domain.com" and "routes" is an array of all routes.

Problem solved. A solution is

{% raw %}{%{% endraw %} set some_url = domain ~ url(['for': 'some-action', 'param1': '123', 'param2': '456']) {% raw %}%}{% endraw %}
