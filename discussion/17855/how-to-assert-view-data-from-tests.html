$response = $this->application->handle('/');

I'm able to make it a post request by overriding the $_SERVER variables.

so the solution is:

$_SERVER['REQUEST_METHOD'] = 'POST';
$response = $this->application->handle('/');

$this->assertContains('test variable', $this->di->getView()->getParamsToView());
