I installed a success ,Linux compiler installation phalcon (512M memory)

  1. sudo yum install php-devel pcre-devel gcc make
  2. git clone --depth=1 git://github.com/phalcon/cphalcon.git
  3. cd cphalcon/ext
  4. /opt/local/php5/bin/phpize
  5. ./configure --with-php-config=/opt/local/php5/bin/php-config
  6. make && make install
  7. vim /opt/local/php5/etc/php.ini
  8. add a row extension=phalcon.so

Solution for ubuntu 12.04 with 512 MB RAM php 5.6. All comand from root

  1. git clone --depth 1 --branch phalcon-v2.0.3 https://github.com/phalcon/cphalcon.git
  2. cd cphalcon/ext
  3. phpize
  4. ./configure --with-php-config=/usr/bin/php-config
  5. make && make install
  6. touch /etc/php5/mods-available/phalcon.ini
  7. echo -e "extension=phalcon.so" >> /etc/php5/mods-available/phalcon.ini
  8. ln -s /etc/php5/mods-available/phalcon.ini /etc/php5/apache2/conf.d/100-phalcon.ini
  9. ln -s /etc/php5/mods-available/phalcon.ini /etc/php5/cli/conf.d/100-phalcon.ini
