I had this length filter issue as well on PHP 7.1 and Phalcon 3.2.2.

I found that {% raw %}{%{% endraw %} if object is not empty {% raw %}%}{% endraw %} works and let's you continue using the object rather than an array.

Just for the sake of safety, I now use {% raw %}{%{% endraw %} if object is defined and object is not empty {% raw %}%}{% endraw %} and it performs entirely as expected.
