Yep that was it. The below code works for stopping as of 3.2


$eventsManager = new EventsManager();
$eventsManager->attach(
    'micro:beforeExecuteRoute',
    function (Event $event, $app) {
        // LOGIC
        return $isAuthed;
    }
);
