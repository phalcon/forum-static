Hi, i am using the dispatcher and forward

$this->dispatcher->forward(
    array(
        "action" => "search",
        "params" => array(1, 2, 3)
    )
);

docs : https://docs.phalcon.io/en/latest/reference/dispatching.html#forwarding-to-other-actions
