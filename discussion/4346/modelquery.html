$q = new \Phalcon\Mvc\Model\Query("SELECT * FROM Users", $this->getDI()); // Users model class
$result = $q->execute();
foreach ($result as $row) {
    echo "Name: ", $row->name, "<br>";
    echo "Email: ", $row->email, "<br>";
}
exit;

(Edit: Please ignore this part if it doesn't apply) Since you are using the limit, group by clauses, you could also use the find() method in Phalcon\Mvc\Model (http://docs.phalcon.io/en/latest/api/Phalcon_Mvc_Model.html). There is a separate static count() method in there too.
