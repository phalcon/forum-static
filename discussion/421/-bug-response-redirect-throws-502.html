If the developer did not pass a http status code it uses 302: https://github.com/phalcon/cphalcon/blob/master/ext/http/response.c#L539

If you run something like this:

$r = new Phalcon\Http\Response();
$r->redirect('http://www.domain.com/path', true);
var_dump($r->getHeaders());

You'll see:

object(Phalcon\Http\Response\Headers)#2 (1) { ["_headers":protected]=> array(3) { ["HTTP/1.1 302 Redirect"]=> NULL ["Status"]=> string(12) "302 Redirect" ["Location"]=> string(26) "http://www.domain.com/path" } }
