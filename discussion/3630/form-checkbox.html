$rememberMe = new Check('remember_me', array(
    'value' => 1,
    'class' => 'box'
));
$this->add($rememberMe);

when you create a new form element you can pass it attributes as array.

http://docs.phalcon.io/en/latest/api/Phalcon_Forms_Element_Check.html
