Yeah, either add it to volt, or just use plain PHP - that's what I do sometimes:

{% raw %}{{{% endraw %} set myArray = ['red','orange','yellow','green','blue','indigo','purple'] {% raw %}}}{% endraw %}

<?php $myNewArray = array_chunk($myArray,2); ?>

{% raw %}{%{% endraw %} for chunk in myNewArray {% raw %}%}{% endraw %}
//code here
{% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
