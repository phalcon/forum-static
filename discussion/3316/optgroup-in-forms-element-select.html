Hello Sir,

you can add optgroups in your form via:

<?php
// ...
$this->add(new Select("status", array(
     "Active" => array('A1' => 'A One', 'A2' => 'A Two'), // "Active" = OptGroupTitle
     "B" => "B One"
)));

source: https://github.com/phalcon/cphalcon/pull/1463
