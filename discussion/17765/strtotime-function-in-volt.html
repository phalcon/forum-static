How do you use it? Functions should be used like so: {% raw %}{{{% endraw %} functionName(variable) {% raw %}}}{% endraw %}.

If you want to use it as a modifierd: {% raw %}{{{% endraw %} variable|functionName {% raw %}}}{% endraw %} add it like so: $compiler->addFilter('strtotime', 'strtotime');
