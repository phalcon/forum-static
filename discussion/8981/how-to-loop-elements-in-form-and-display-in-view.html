Hi guys,

Is there a way where i can loop all my Elements text fields, select dropdowns, buttons from my Form


class SampleForm extends Form{
    public function initialize(){

        $fname = new Text('fname');
        $lname = new Text('lname');
        $mname = new Text('mname');

        $this->add($fname, $lname, $mname);
    }
}

and in my form view I can do something like


{% raw %}{%{% endraw %} for elements in  form {% raw %}%}{% endraw %}
    <div class="form-group">
        {% raw %}{{{% endraw %} elements {% raw %}}}{% endraw %}
     </div>
{% raw %}{%{% endraw %} endfor  {% raw %}%}{% endraw %}

is this possible?

No. You add them independently. 

    $this->add($fname);
    $this->add($lname);
    // ...

In your controller:

    $form = new YourForm();
    $this->view->form = $form;

And what you can do in your template is:

    {% raw %}{%{% endraw %} for element in form {% raw %}%}{% endraw %}
            <div class="form-group">
                <label for="{% raw %}{{{% endraw %} element.getName() {% raw %}}}{% endraw %}">{% raw %}{{{% endraw %} element.getLabel() {% raw %}}}{% endraw %}</label>
                {% raw %}{{{% endraw %} element {% raw %}}}{% endraw %}
            </div>  
    {% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
