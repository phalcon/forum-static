This syntax is not supported because it's a MySQL extension that cannot be translated by PHQL when using PostgreSQL/SQlite/Oracle. You can use a raw sql query in this case: http://docs.phalcon.io/en/latest/reference/phql.html#using-raw-sql

Implementation is different in PostgreSQL:

INTERVAL '100 days'
INTERVAL '2 months'

vs mysql:

INTERVAL 100 DAY
INTERVAL 2 MONTH

In Oracle, Interval is used to convert a date/datetime/time into another date/datetime/time:

INTERVAL ' 11:12:10.1234567' hour to second
