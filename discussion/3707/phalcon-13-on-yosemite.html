I found the solution.

OS X 10.10 come with gcc 4.8.3
when try to install phalcon I found error

gcc: warning: couldn't understand kern.osversion '14.0.0

The solution is

  • upgrade gcc to newer version (4.9.1)
  • re-install phalcon
