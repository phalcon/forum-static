You can use the \DateTime object from PHP (https://php.net/manual/en/datetime.modify.php) and then use:

$date = new \DateTime();
// modify date ...
$this->tag->setDefault('myDate', (string) $date);
echo Phalcon\Tag::dateField('myDate');
