I have a macro that looks like:

{% raw %}{%{% endraw %}- macro image_url(pic_id, big) {% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %} if big == true {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} set prefix = 2 {% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %} else {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} set prefix = 1 {% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %} endif {% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %} return "https://example.com/images/" ~ prefix ~ "_" ~ pic_id ~ ".jpg" {% raw %}%}{% endraw %}
{% raw %}{%{% endraw %}- endmacro {% raw %}%}{% endraw %}

...and I call this macro as:

<a href="{% raw %}{{{% endraw %}image_url(1234, true){% raw %}}}{% endraw %}">

But the resulting HTML shows up as:

<a href="        https://example.com/images/2_1234.jpg">

Why all these extra spaces???

You might want to remove extra spaces in the macro:

{% raw %}{%{% endraw %}- macro image_url(pic_id, big) -{% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %}- if big == true -{% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %}- set prefix = 2 -{% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %}- else -{% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %}- set prefix = 1 -{% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %}- endif -{% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %}- return "https://example.com/images/" ~ prefix ~ "_" ~ pic_id ~ ".jpg" -{% raw %}%}{% endraw %}
{% raw %}{%{% endraw %}- endmacro -{% raw %}%}{% endraw %}
