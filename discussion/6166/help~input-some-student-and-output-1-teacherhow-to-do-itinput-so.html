you may use query bulder:


$teachers = $this->modelsManager->createBuilder()
    ->from('StudendsTeachers')
     ->where('student_id = 1')
     ->andWhere('student_id = 2');
    ->getQuery()
    ->execute();

btw, I'm not sure that you will get an unique record with the kind of relations you are using ...
