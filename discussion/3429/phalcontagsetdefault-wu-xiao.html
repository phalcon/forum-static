You can use something like this:

In controller: 

        $tag = new \Phalcon\Forms\Element\Text('iput2');
        $tag->setDefault('test');

        $this->view->iput2 = $tag;

In view (volt):

    {% raw %}{{{% endraw %} iput2.render() {% raw %}}}{% endraw %}

Or with tags:

In controller: 

    \Phalcon\Tag::setDefault("ipu2", "test");

In volt:

    {% raw %}{{{% endraw %} text_field("ipu2") {% raw %}}}{% endraw %}
