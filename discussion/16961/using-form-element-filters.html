I found solution, but is this the best approach to filters input values? I cant do it in more simpler way?

    $postData = [];

    foreach ($loginForm as $element) {
        $postData[$element->getName()] = $this->request->getPost(
            $element->getName(),
            $element->getFilters()
        );
    }
