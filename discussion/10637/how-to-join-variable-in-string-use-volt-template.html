Hi, I have a declared value in the controller named: {% raw %}{{{% endraw %} themeUrl {% raw %}}}{% endraw %}. I want to Join this variable to string in Volt template it works well with php: <?php $this->assets->collection('footer')->addJs($themeUrl.'/js/icheck.min.js');?>

But with Volt tempalte is Error: {% raw %}{%{% endraw %} do assets.collection('footer').addJs({% raw %}{{{% endraw %} themeUrl {% raw %}}}{% endraw %}'/js/icheck.min.js') {% raw %}%}{% endraw %}} Please help me

The ~ concatinates strings in volt and you have too many curly bracers around themeUrl

addJs(themeUrl ~ '/js/icheck.min.js')
