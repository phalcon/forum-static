Page is composed from hierarchical view layers. Every higher layer should use below methods to render lower layers
Everything is clearly described here http://docs.phalcon.io/en/latest/reference/views.html

in phtml 

<?php echo $this->getContent() ?>

in volt 

{% raw %}{{{% endraw %} content() {% raw %}}}{% endraw %}
