The only way it returns false when:

  • https://github.com/phalcon/cphalcon/blob/2.0.0/phalcon/mvc/model.zep#L2782, _preSave returns false because of an automatic or user-defined validation returns false
  • https://github.com/phalcon/cphalcon/blob/2.0.0/phalcon/mvc/model.zep#L2805 _postSave related records cannot be save
