Is there some way to use {% raw %}{%{% endraw %} block... {% raw %}%}{% endraw %} in link_to(). I am trying to use template inheritence. The below code adds literal {% raw %}{%{% endraw %} block indexlink {% raw %}%}{% endraw %}{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %} into the link

<li class="previous">{% raw %}{{{% endraw %} link_to("{% raw %}{%{% endraw %} block indexlink {% raw %}%}{% endraw %}{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}", "Go Back") {% raw %}}}{% endraw %}</li>

If I remove the double quotes then the compiler gives an error - unexpected token {% raw %}{%{% endraw %}. If I remove {% raw %}{%{% endraw %} blockand {% raw %}%}{% endraw %} totally then it says Undefined variable: indexlink. Is there some way I can specify the link using a {% raw %}{%{% endraw %} block... variable {% raw %}%}{% endraw %}option.

No, but you can do this:

{% raw %}{%{% endraw %} block indexLink {% raw %}%}{% endraw %}
{% raw %}{%{% endraw %} set indexLink = '/mylink' {% raw %}%}{% endraw %}
{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}

<li class="previous">{% raw %}{{{% endraw %} link_to(indexLink, "Go Back") {% raw %}}}{% endraw %}</li>

Or you could propagate a variable from the controller.
