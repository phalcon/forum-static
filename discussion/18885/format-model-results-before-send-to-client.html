You have to enable following PDO options when connection to your database:

'options' => [
\PDO::ATTR_EMULATE_PREPARES => false,
\PDO::ATTR_STRINGIFY_FETCHES => false,
],

This allows Phalcon (i.e. PDO) to convert numeric values to INT
