If your checkbox name in same name, you can try like this

In View Volt :

{% raw %}{%{% endraw %} for category in categories {% raw %}%}{% endraw %}

<input type="checkbox" value="{% raw %}{{{% endraw %} category.id {% raw %}}}{% endraw %}" name="category[]">

{% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}

In Controller :

$cats=$this->request->getPost('category');

foreach ($cats as $cat){

echo $cat;

}
