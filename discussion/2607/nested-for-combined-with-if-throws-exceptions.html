Hi there,

I have meet a strange behavior in volt. I have this:

        {% raw %}{%{% endraw %} if some_var {% raw %}%}{% endraw %}
            {% raw %}{%{% endraw %} for item in some_var['items'] {% raw %}%}{% endraw %}
                {% raw %}{%{% endraw %} for object in item {% raw %}%}{% endraw %}
                    {% raw %}{%{% endraw %} include 'some/partial' with {'object':object} {% raw %}%}{% endraw %}
                {% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
            {% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} else{% raw %}%}{% endraw %}
            Nothing here
        {% raw %}{%{% endraw %} endif {% raw %}%}{% endraw %}

Throws an exception : Phalcon\Mvc\View\Exception: Unexpected ENDIF

If i remove the second FOR, this works:

        {% raw %}{%{% endraw %} if some_var {% raw %}%}{% endraw %}
            {% raw %}{%{% endraw %} for item in some_var['items'] {% raw %}%}{% endraw %}
                // Do  some stuff
            {% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} else{% raw %}%}{% endraw %}
            Nothing here
        {% raw %}{%{% endraw %} endif {% raw %}%}{% endraw %}

Any idea ?
