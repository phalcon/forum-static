Just checked - Phalcon 2.0.0 is available for php 5.3, 5.4, 5.5 and 5.6 on brew. Possible you need to update your brew formulas with brew update and after this try to install phalcon package with instructions from phalcon.io or update existing packages with brew upgrade --all. P.S. You can also update only phalcon package brew upgrade php56-phalcon or replace 56 with your php version. But i recommend you to upgrade everything.
