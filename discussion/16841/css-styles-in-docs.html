Firefox displays everything ok. time to use the diagnostics in your Web browser.

1/ A CSS file might not load. In the Firefox debug stuff, you can see a list of download files. Have a look at the equivalent in your Web browser.

2/ If all the files loaded, open up the CSS inheritance debug. In Firebug, this is displayed by right clicking the element and choosing Inspect Element. Your browser might choose a different CSS rule or default.
