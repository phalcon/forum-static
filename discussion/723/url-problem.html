My site url: http://localhost/phalconphp

I've created an index page where it shows "Signup here" hyperlink. When I clicked on the link, it goes to "http://localhost/public/signup" instead of "http://localhost/hellophalcon/public/signup"

Add in you index.php:

  $di = new Phalcon\DI\FactoryDefault();
.....
/**
     * The URL component is used to generate all kind of urls in the application
     */
    $di->set('url', function() use ($config){
        $url = new \Phalcon\Mvc\Url();
        $url->setBaseUri( '/phalconphp/' ); // or $url->setBaseUri( 'http://localhost/phalconphp' );
        return $url;
    });
