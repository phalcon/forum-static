"5" will be passed to your action as the first argument. Your controller action needs to set that value in the view, as a variable (for example, user_id). You can then access "5" in Volt by accessing user_id. For example, {% raw %}{{{% endraw %} user_id {% raw %}}}{% endraw %}.

There are many ways to access URL parameters in view but the best practice is that you should get it from dispatcher and pass it to view via controller/action

a basic example:

// URL : http://www.abc.com/user/profile/5

// in action
$this->view->user_id = $this->dispatcher->getParams()[0];
// in view
{% raw %}{{{% endraw %} user_id {% raw %}}}{% endraw %}

Or directly in volt file

{% raw %}{%{% endraw %} var params = dispatcher.getParams() {% raw %}%}{% endraw %}
{% raw %}{{{% endraw %} params[0] {% raw %}}}{% endraw %}
