I found the error, the router comes into conflict with arguments that have default numeric..

exemplo

 $this->add (
            '/controller/:action/:int/:params',
            array (
                'action' => 1,
                'id'     => 2,
                'page' => 1 //value default 1 conflicts with action 1
            )
        )
        ->setName ( 'teste' );
