Yeah this worked for me too. The issue is that phalcon 1.3.2 is NTS but WAMP is TS. Only the older version is TS. Older version which is TS is Phalcon 1.3.1 - Windows x64 for PHP 5.5.0 (VC11) (March 26 2014 18:59:48 EDT / sha1: 48f41f6d3b8de3b5d3643a826dea696c49bdcd52) which could be downloaded easy from http://phalcon.io/en/download/windows .
