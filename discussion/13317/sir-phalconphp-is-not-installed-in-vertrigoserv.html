Such all-in-one packages are only for basics, in general. When you need to add an extension it can turn into a mess.

Setup VirtualBox or similar software, and run full GNU/Linux with Phalcon. I know this sounds too cumbersome for many devs out there, but if you plan to develop some serios apps with Phalcon, your production environment is going to be GNU/Linux anyway, so it's best to save yourself some troubles when migrating from dev to production env.
