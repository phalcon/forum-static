You can use the Element::setUserOption() method to set the values and the specific parameters you want to use. In Volt template, you have to generate it by the list of your values (through a loop). Sth like :


{% raw %}{%{% endraw %} set boolIsRadio = objField.getUserOption('is_unique') {% raw %}%}{% endraw %}
{% raw %}{%{% endraw %} set arrayCheckMultiple = objField.getUserOption('multiple') {% raw %}%}{% endraw %}

{% raw %}{%{% endraw %} for strCheckKeyName, strLabel in arrayCheckMultiple {% raw %}%}{% endraw %}
             {% raw %}{%{% endraw %} set strCurrentFieldName = '[' ~ strCheckKeyName ~ ']' {% raw %}%}{% endraw %}
             <label>
            {% raw %}{%{% endraw %} if boolIsRadio {% raw %}%}{% endraw %}
               {% raw %}{{{% endraw %} radio_field(objField) {% raw %}}}{% endraw %}
            {% raw %}{%{% endraw %} else {% raw %}%}{% endraw %}
                {% raw %}{{{% endraw %} check_field(objField, 'name':strCurrentFieldName) {% raw %}}}{% endraw %}
            {% raw %}{%{% endraw %} endif {% raw %}%}{% endraw %}
            {% raw %}{{{% endraw %} strLabel {% raw %}}}{% endraw %}
            </label>
{% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}

(it assumes here you are looping on the form's fields)
