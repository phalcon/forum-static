Hey man

We have two solution for problem of you.

Case1: We have assume url login is /login/username to login is contoller

    {% raw %}{%{% endraw %} if this.view.getControllerName() == 'login'{% raw %}%}{% endraw %}
        //do hide code
    {% raw %}{%{% endraw %}else{% raw %}%}{% endraw %}
      //do display code
    {% raw %}{%{% endraw %}endif{% raw %}%}{% endraw %}

Case 2: If you set session for user as login:

    {% raw %}{%{% endraw %} if ! this.get.seesion('auth') {% raw %}%}{% endraw %}
        //do hide code
    {% raw %}{%{% endraw %}else{% raw %}%}{% endraw %}
      //do display code
    {% raw %}{%{% endraw %}endif{% raw %}%}{% endraw %}

<html>
<title></title>
