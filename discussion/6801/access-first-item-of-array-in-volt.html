Using {% raw %}{{{% endraw %} arrayList[0].id {% raw %}}}{% endraw %} will produce ($arrayList[0])->id in the php code generated in volt - causing the error you saw.

This can be overcome by setting the first item in the array to a variable and then accessing it:

{% raw %}{%{% endraw %} set firstItem = arrayList[0] {% raw %}%}{% endraw %}
{% raw %}{{{% endraw %} firstItem->id {% raw %}}}{% endraw %}

Not as elegant as you were trying, but gets around how volt is parsed.

Arrays:

  {% raw %}{{{% endraw %} arrayList[0]['id'] {% raw %}}}{% endraw %} // translates to arrayList[0]['id']

objects

  {% raw %}{{{% endraw %} arrayList.[0].id {% raw %}}}{% endraw %} // translates to $arrayList[0]->id

try using the dump feature as suggested to see the output.
