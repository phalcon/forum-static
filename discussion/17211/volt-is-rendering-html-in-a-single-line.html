Hi @mauro other simple way using {% raw %}{{{% endraw %}- -{% raw %}}}{% endraw %} and {% raw %}{%{% endraw %}- -{% raw %}%}{% endraw %} 

{% raw %}{%{% endraw %}- if categories is defined -{% raw %}%}{% endraw %}
    {% raw %}{{{% endraw %}- "<h1>Categories</h1>" -{% raw %}}}{% endraw %}
    {% raw %}{{{% endraw %}- "<ul>" -{% raw %}}}{% endraw %}
    {% raw %}{%{% endraw %}- for category in categories -{% raw %}%}{% endraw %}
        <li>
        {% raw %}{{{% endraw %}- '<span class="label label-default pull-right">' ~ category.number_posts ~ '</span>' ~ category.name -{% raw %}}}{% endraw %}
        </li>
    {% raw %}{%{% endraw %}- endfor -{% raw %}%}{% endraw %}
    {% raw %}{{{% endraw %}- "</ul>" -{% raw %}}}{% endraw %}
{% raw %}{%{% endraw %}- endif -{% raw %}%}{% endraw %}

output

<h1>Categories</h1><ul><li>cat1</li><li>cat2></li></ul>

I hope it's help you. Good luck
