Yea it looks you use the false regex

you use the regex for a action:[a-zA-Z0-9_]+

For a param is (/.*)*

Or you can try to change your action routing to

$router->add(
    "(/.*)*",
    array(
        "controller" => "user",
        "action"     => "profile",
        "param"         => 1,
    )
);
