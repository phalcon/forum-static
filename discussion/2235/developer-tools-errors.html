I've found the solution

In file C:\xampp\htdocs\phalcon-devtools-1.3.x\scripts\Phalcon\Web\Tools.php Line 217, comment out the

chdir('..');

on Line 229 & 237 change from

$cpath = $configPath . '/config.ini';

to

$cpath = '../'.$configPath . '/config.ini';

that works for me.
