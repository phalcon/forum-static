It won't work. You have to define your routes in bootstrap before use them. 

$di['router']=function() {
   $router = new Phalcon\Mvc\Router();
   $router->add(
                "/",
                array(
                        "controller" => "index",
                        "action"     => "get")
    );
    return $router;
}

If you have to define routes in controllers you have to use Annotations Router. http://docs.phalcon.io/en/latest/reference/routing.html#annotations-router
