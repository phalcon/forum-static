depending on Volt documentation it should be something like this:

{% raw %}{%{% endraw %} set var = "abc" {% raw %}%}{% endraw %}
{% raw %}{%{% endraw %} set var = var ~ "def" {% raw %}%}{% endraw %}
{% raw %}{%{% endraw %} set var = var ~ "ghi" {% raw %}%}{% endraw %}
