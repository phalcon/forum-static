There is an autoescape mode

https://docs.phalcon.io/en/latest/reference/volt.html#autoescape-mode

//Manually escaped: 
{% raw %}{{{% endraw %} robot.name|e {% raw %}}}{% endraw %}

// Auto escaped:
{% raw %}{%{% endraw %} autoescape true {% raw %}%}{% endraw %}
{% raw %}{{{% endraw %} robot.name {% raw %}}}{% endraw %}
{% raw %}{%{% endraw %} endautoescape {% raw %}%}{% endraw %}
