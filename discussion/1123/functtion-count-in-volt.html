You'll can get this value easy with loop controls : https://docs.phalcon.io/en/latest/reference/volt.html#loop-context Example :

{% raw %}{%{% endraw %} set a = [1,2,3] {% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %} for val in a {% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %} if loop.first {% raw %}%}{% endraw %}
        {% raw %}{{{% endraw %} loop.length {% raw %}}}{% endraw %}
    {% raw %}{%{% endraw %} endif {% raw %}%}{% endraw %}
    {% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
