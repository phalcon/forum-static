echo isset($_GET['param']) ? 'class="active"' : '';

How would I go about (a) getting this parameter with Volt, and (b) recreating that line of code with Volt syntax?

Current code:

{% raw %}{%{% endraw %} for (item in object) {% raw %}%}{% endraw %}
    <li>{% raw %}{{{% endraw %} link_to('index/browse/' ~ item.id, '<i class="fa fa-caret-right"></i>' ~ item.id) {% raw %}}}{% endraw %}</li>
{% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}

Hey friend :)

{% raw %}{%{% endraw %} if request.has('param') {% raw %}%}{% endraw %}
    {% raw %}{{{% endraw %} request.get('param') {% raw %}}}{% endraw %}
{% raw %}{%{% endraw %} endif {% raw %}%}{% endraw %}

Or the shorthand version:

 {% raw %}{{{% endraw %} request.has('param') ? "hello" : "not-defined" {% raw %}}}{% endraw %}
