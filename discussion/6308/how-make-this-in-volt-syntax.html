Maybe you can use an alternative url('link/something'):

<a href="{% raw %}{{{% endraw %} url('link/something')  {% raw %}}}{% endraw %}" class="my-class"><span class="my-class"></span></a>
<a href="{% raw %}{{{% endraw %} url('link/something')  {% raw %}}}{% endraw %}"><img alt="" src="my_image.jpg"></a>
