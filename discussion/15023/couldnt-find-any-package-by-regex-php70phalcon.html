hi,

sudo apt-get install build-essential php7.0-dev libpcre3-dev

git clone git://github.com/phalcon/cphalcon.git

cd cphalcon/build

sudo ./install

sudo nano /etc/php/7.0/apache2/conf.d/30-phalcon.ini 

    put in it:  extension=phalcon.so

sudo nano /etc/php/7.0/cli/conf.d/30-phalcon.ini 

    put in it:  extension=phalcon.so

sudo service apache2 restart

you're done

PS: made an edit because of the wrong path for 30-phalcon.ini
