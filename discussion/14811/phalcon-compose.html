This is an unofficial, open-source and community-driven boilerplate for Phalcon projects that run on Docker. It's an attempt of standardizing and making it easier to bootstrap Phalcon applications ready for development and production environments. The main tools used are Phalcon, Docker and Docker Compose.

https://phalcon-compose.readme.io/
