Is it possible to do something like:

{% raw %}{%{% endraw %} set test = include 'parts/flight-search-form' {% raw %}%}{% endraw %}

Or even something like:

{% raw %}{{{% endraw %} str_replace(
    '[flight-search-form]',
    partial("parts/flight-search-form", ['searchType': 'simple']),
    obj.content
) {% raw %}}}{% endraw %}

Second example always renders (echo's) the partial even tho if the shortcode is not present.

This should do it ($this->view->getPartial()):

{% raw %}{{{% endraw %} str_replace('[flight-search-form]', view.getPartial('parts/flight-search-form', ['searchType':'simple']), obj.content) {% raw %}}}{% endraw %}
