Seems to be a problem within your Code, PHP- or Phalcon-Installation. Turn on you PHPs error_reporting and look at your php-logs

Try adding some output and exit in the front of your index.php. i.e.

<?php
echo "Success";
exit;
//...

If the request of http://SERVER_IP_ADDR/test-api/api/robots now returns "Success", PHP is up and running.

then try

<?php

$app = new Phalcon\Mvc\Micro();

echo "Success";
exit;
//...

If the request of http://SERVER_IP_ADDR/test-api/api/robots now returns "Success", Phalcon is up and running, too.
