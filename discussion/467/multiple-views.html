You can integrate another view into your volt template https://docs.phalcon.io/en/latest/reference/volt.html#view-integration For header and footer it's better to include them in views/index.volt, as they are common for all pages. For example (index.volt):

{% raw %}{{{% endraw %} partial('shared/header') {% raw %}}}{% endraw %} {% raw %}{{{% endraw %} content() {% raw %}}}{% endraw %} {% raw %}{{{% endraw %} partial('shared/footer') {% raw %}}}{% endraw %}

You can use blocks. In your index.volt :

<html>
    <body>
        {% raw %}{%{% endraw %} block page_header {% raw %}%}{% endraw %}{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} block page_body {% raw %}%}{% endraw %}{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} block page_footer {% raw %}%}{% endraw %}{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}
    </body>
</html>

Then, in edit.volt for example:

{% raw %}{%{% endraw %} extends 'index.volt' {% raw %}%}{% endraw %}

{% raw %}{%{% endraw %} block page_header {% raw %}%}{% endraw %}
    Page header
{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}

{% raw %}{%{% endraw %} block page_body {% raw %}%}{% endraw %}
    Page content
{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}

{% raw %}{%{% endraw %} block page_footer {% raw %}%}{% endraw %}
    Page footer
{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}
