If i need create a SELECT with authors ('id' as value and 'lastname' and 'firstname' as name) i use: {% raw %}{{{% endraw %} select('author', author, 'using': ['id', 'lastname']) {% raw %}}}{% endraw %}

Create the query joining the lastname with the firstname:

$this->view->authors = 
       $this->modelsManager->createQuery()
           ->columns('id, CONCAT(lastname, " ", firstname) AS realname')
           ->from('Authors')

Then in the View:

{% raw %}{{{% endraw %} select('author', author, 'using': ['id', 'realname']) {% raw %}}}{% endraw %}
