This should work:

<?php
$DI->set('router',function() use($Config){
    $Router = new \Phalcon\Mvc\Router();
    $Router->notFound(array(
        "controller" => "index",
        "action" => "route404"
    ));
    return $Router;
});

The default route rules that Phalcon\Mvc\Router() have should still be in effect.
