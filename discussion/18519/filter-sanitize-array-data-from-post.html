It works for array values also. Just tested myself.

Without sanitize $this->request->getPost('asd')

Array
(
    [0] => asd___1
    [1] => asd___5
    [2] => asd___8
)

With sanitize $this->request->getPost('asd', 'int')

Array
(
    [0] => 1
    [1] => 5
    [2] => 8
)
