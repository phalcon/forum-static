Right now we do not have a ready to use package builder for Kali Linux. You can use the standard way to build from source:

With Zephir

git clone --depth=1 -q [email protected]:phalcon/cphalcon.git
cd cphalcon

zephir fullclean
zephir build

Without Zephir

git clone --depth=1 -q [email protected]:phalcon/cphalcon.git
cd cphalcon

cd build
sudo ./install
