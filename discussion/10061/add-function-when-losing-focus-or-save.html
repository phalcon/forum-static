Knowing the error message reported by Phalcon would be helpful, I don't think anyone will take the time to debug your code.

Phalcon models have built-in validation logic, you should use it: https://docs.phalcon.io/en/latest/reference/models.html#implementing-a-business-rule

If you want to validate HTML forms on the fly (when an input loses focus), you'll have to send an AJAX request to the server for evaluation.
