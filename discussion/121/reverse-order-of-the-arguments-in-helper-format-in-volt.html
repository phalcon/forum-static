Hi, I have a question about helpers "format": I expected it to be used as "{% raw %}{{{% endraw %} (item.total_amount_uah/item.dollar_rate)|format('%.2f') {% raw %}}}{% endraw %}" and turned you so "{% raw %}{{{% endraw %} '%.2f'|format(item.total_amount_uah/item.dollar_rate){% raw %}}}{% endraw %}", it is a mistake?

No it is a convention in Volt where the left operand of the pipe ('|') operator is the first argument of the related php function. Format uses sprintf, and the 1st sprintf argument is the string format, so that the left operand of the pipe is the format.
