How it's possible to set an alias in the query select using QueryBuilder ? For example :

Andres Gutierrez

This way

$robots = $this->modelsManager->createBuilder()
    ->columns(array("tid" => "id", "tname" => "name"))
    ->from('Robots')
    ->getQuery()
    ->execute();
