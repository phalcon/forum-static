Version 2.0.13 stable was latest in 2.0.x series, so you should use that one.

2.0.x series DLL files were always published at: https://phalcon.io/en/download/windows

But now they're gone as that release is non-LTS and no one shall be using that for new projects.

Bottomline: if you really need to maintain old 2.0.x project(s), why don't you simply use GNU/Linux version instead, much less hassle than Windows?
