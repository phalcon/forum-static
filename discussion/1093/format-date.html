just write

{% raw %}{{{% endraw %} date("d-m-Y") {% raw %}}}{% endraw %}

or

{% raw %}{{{% endraw %} date("d-m-Y", time()) {% raw %}}}{% endraw %}

If you want print custom time from variable, this variable must be integer type of Unix timestamp. Example

{% raw %}{%{% endraw %} set today = time() {% raw %}%}{% endraw %}
<p>Today is: 
{% raw %}{{{% endraw %} date("d-m-Y", today) {% raw %}}}{% endraw %}
</p>

{% raw %}{%{% endraw %} set tomorrow = time()+86400 {% raw %}%}{% endraw %}
<p>Tomorrow is: 
{% raw %}{{{% endraw %} date("d-m-Y", tomorrow) {% raw %}}}{% endraw %}
</p>
