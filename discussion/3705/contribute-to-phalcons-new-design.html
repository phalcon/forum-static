Some weeks ago, we started a contest to choose the new design of Phalcon in its second major version. Four great proposals were submitted. Using a survey the community have chosen their favorite design. Many thanks to the all the participants for their enthusiasm. Each design provides interesting ideas that should be taken in the final design.

These are the results:

# Author Votes
1. Roberto #1 357
2. Max 311
3. Roberto #2 102
4. Theoretic 97
867

Congratulations to @fogcity, his design has been choosen for Phalcon 2.

What's next? We encourage the community to help Roberto out implementing this design in the main site. Regarding us, you have all our support on anything you need to implement this design.

Thanks

Hey, the new homepage static layout is up, merged into website/redesign.2.0.

For the record:

  • The layout is based on Twitter Bootstrap 3.1 and jQuery (latest)
  • Backstretch - header image stretch
  • Highlight.js - code samples highlight
  • Magnific Popup - vimeo video popup
  • Icomoon - custom icons embedded in font
  • FamFamFam - flag icons
  • Google Fonts (Open Sans)

The homepage still needs polishing. Later I'll add the HTML5-header animation and focus on sub-page layout.
