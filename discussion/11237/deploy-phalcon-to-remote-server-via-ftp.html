Prerequisities:

  1. PHP 5.x or 7.0 (ideally the same you use on your machine)
  2. Phalcon extension which you need (1.3, 2.0 or 2.1)
  3. PHP extensions which you use (gd, pdo, curl, ...)

other software like DB, webserver, etc.
