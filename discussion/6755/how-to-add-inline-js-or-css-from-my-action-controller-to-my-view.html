You can try this:

In the controller

$code_js = 'alert("hola");';

$this->assets->collection('jsVar')->addInlineJs('$(document).ready(function() {' . $code_js . '});');

and in the view

{% raw %}{{{% endraw %} assets.outputInlineJs ('jsVar') {% raw %}}}{% endraw %}

Hope this help!
