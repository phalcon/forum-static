Just do it like:


  $parameters = [
                    "name" => "%{$name}%"
                ];

$foundedNames = Name::findFirst(
                    [
                        "name LIKE :name:",
                        "bind" => $parameters
                    ]
                );

To further explain - you cannot put a bound parameter in a like statement, surrounded by '%' - the sql compiler doesn't like that. So, you need to add the '%' to the value that gets bound.
