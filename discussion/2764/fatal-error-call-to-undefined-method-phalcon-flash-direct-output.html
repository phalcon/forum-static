Hi.

You are using Direct flash messages. This class does not have an output() method. Check here: http://docs.phalcon.io/en/latest/api/Phalcon_Flash_Direct.html. There are 6 methods you can output message but unfortunetly not output().

On the other hand output() method exist in Session flash class Phalcon\Flash\Session: http://docs.phalcon.io/en/latest/api/Phalcon_Flash_Session.html.
