Ok i have fixed it .. It was a wrong place where I put this charset option. Since Im usig Phalcon PDO .. i had to put it in there, not in an external config file..

        "options" => array(
            PDO::MYSQL_ATTR_INIT_COMMAND => 'SET NAMES utf8'
        )
