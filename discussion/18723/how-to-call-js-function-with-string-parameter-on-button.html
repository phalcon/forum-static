I think you have enabled Volt autoescape. Try like this:

{% raw %}{%{% endraw %} autoescape false {% raw %}%}{% endraw %}
{% raw %}{{{% endraw %} submit_button("Добавить", "class":"btn btn-primary btn-sm", "onclick": "alert('cost')") {% raw %}}}{% endraw %}
{% raw %}{%{% endraw %} endautoescape {% raw %}%}{% endraw %}
