Hi all,

i don't know if it is a bug but i'm not able to call any method over an object stored in array: The following code compiles in a wrong PHP code wich won't work: {% raw %}{{{% endraw %} my_array['my_object'].my_method() {% raw %}}}{% endraw %}

The result of the compilation is like: <?php echo ($my_array['my_object'])->my_method(); ?>

As you can see the extra parentheses surrounding $my_array['my_object'] will result in a PHP Fatal Error.

As a workaround i've the need every time to use a temp var like this {% raw %}{%{% endraw %} set my_obj = my_array['my_object'] {% raw %}%}{% endraw %} {% raw %}{{{% endraw %} my_obj.my_method() {% raw %}}}{% endraw %}

I suppose VOLT should not add those parentheses so far. Thanks
