Don't forget to run 'make test'.

It's not a Phalcon's message. It's a standard message generated by PHP. If you want to run tests you have to prepare environment and satisfy dependencies. For more see:

  • https://github.com/phalcon/cphalcon/tree/3.4.x/tests
  • https://github.com/phalcon/cphalcon/blob/3.4.x/.travis.yml
