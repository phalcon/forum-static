Phalcon 1.2.3 using:

{% raw %}{{{% endraw %} image(img,false) {% raw %}}}{% endraw %}

where

img="http://lorempixel.com/1000/400/sports/1"

generates:

<img src="/phalcon/foundation/http://lorempixel.com/1000/400/sports/1">

I saw an issue on github (https://github.com/phalcon/cphalcon/issues/1158) saying that this was fixed in 1.3.0 branch. But what for stable 1.2.3 ?
