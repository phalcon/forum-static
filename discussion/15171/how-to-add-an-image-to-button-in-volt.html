Hi,

something like this will do the trick:

                {% raw %}{{{% endraw %} tag_html("button", ["type":"submit","class": "btn btn-primary"]) {% raw %}}}{% endraw %}
                Edit <i class='glyphicon glyphicon-pencil'></i>
                {% raw %}{{{% endraw %} tag_html_close("button") {% raw %}}}{% endraw %}
