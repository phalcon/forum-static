I think you need to set up a base template

view/base.volt:

<html>
<head>
bootstrap js and css goes here
</head>
<body>
{% raw %}{%{% endraw %} block content {% raw %}%}{% endraw %}{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}
</body>
</html>

view/index/index.volt:

{% raw %}{%{% endraw %} extends "base.volt" {% raw %}%}{% endraw %}

{% raw %}{%{% endraw %} block content {% raw %}%}{% endraw %}
<div>information</div>
{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}

Now you should be able to see how to add a footer, your menu can stay in the base.volt unless you have variable menus.

Please see inheritance: http://docs.phalcon.io/en/latest/reference/volt.html#multiple-inheritance
