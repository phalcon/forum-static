If you are creating model in 2.0.x/editing existing then:

$model->assign([
    'name'=>'name',
    'restrictions'=>[]
]);

If you are creating new model in 2.1.x then it's much simpler:

$model = new SomeModel([
    'name'=>'name',
    'restrictions'=>[]
]);
