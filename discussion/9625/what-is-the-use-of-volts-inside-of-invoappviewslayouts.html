When create view files corresponding to controller/action, use of volt engine is best way for create it easily and fastest. Script in volt is very easy than php plain-text. For instance, {% raw %}{{{% endraw %} content {% raw %}}}{% endraw %} equal to code "<?php echo $this->getContent(); ?>".
