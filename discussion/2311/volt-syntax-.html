Hi @G Wynne,

the only difference about {% raw %}{%{% endraw %}- and {% raw %}{%{% endraw %} I know, is that {% raw %}{%{% endraw %}- creates inline html. For example an unordered list:

<ul id="nav" class="menu">
    {% raw %}{%{% endraw %} for menu in menus {% raw %}%}{% endraw %}
        <li>{% raw %}{{{% endraw %} link_to(menu.url, menu.title|e) {% raw %}}}{% endraw %}</li>
    {% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
</ul>

this would output:

<ul id="nav" class="menu">
    <li><a href="/home">Home</a></li>
    <li><a href="/about">About</a></li>
    <li><a href="/news">News</a></li>
    <li><a href="/vita">Vita</a></li>
</ul>

but this:

<ul id="nav" class="menu">
    {% raw %}{%{% endraw %}- for menu in menus -{% raw %}%}{% endraw %}
        <li>{% raw %}{{{% endraw %}- link_to(menu.url, menu.title|e) {% raw %}}}{% endraw %}</li>
    {% raw %}{%{% endraw %}- endfor -{% raw %}%}{% endraw %}
</ul>

would output:

 <ul id="nav" class="menu"><li><a href="/home">Home</a></li><li><a href="/about">About</a></li><li><a href="/news">News</a></li><li><a href="/vita">Vita</a></li></ul>
