So let's say that I have a volt page like this:

{% raw %}{{{% endraw %} link_to("index/somepage",'<span class="glyphicon someglyph"></span> MyLink ',"class":"btn btn-default") {% raw %}}}{% endraw %}

I want to make the text in the second parameter - the text of the link - to be dynamic. So lets say inside the controller here I have something like $this->view->somevar = "test"; then what I want to do inside the volt is:

{% raw %}{{{% endraw %} link_to("index/somepage",'<span class="glyphicon someglyph"></span> MyLink {% raw %}{{{% endraw %}somevar{% raw %}}}{% endraw %}',"class":"btn btn-default") {% raw %}}}{% endraw %}

Is there any way to accomplish this?

You can concatenate strings inside Volt with the ~ operator:

{% raw %}{{{% endraw %} link_to("index/somepage",'<span class="glyphicon someglyph"></span> MyLink '~somevar~'',"class":"btn btn-default") {% raw %}}}{% endraw %}
