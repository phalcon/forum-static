Hi. Yes, odm supports projection. Use the "fields" key.

class Robots extends \Phalcon\Mvc\Collection
{
    // assume parts is an array of parts
    public $parts;
}

Robots::find([
    'fields' => [
        'parts' => [
            '$slice' => 2 // get first two parts for each robot
        ]
    ]
]);
