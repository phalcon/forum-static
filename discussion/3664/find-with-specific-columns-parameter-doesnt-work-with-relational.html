When you set the 'columns' parameter it returns an incomplete object, this is mentioned in the docs: http://docs.phalcon.io/en/latest/reference/models.html#finding-records

An incomplete object is a partial representation of the model with writing and other features not available due to possible missing data to perform these operations.
