Have you tried setting the lifetime when you use it?

Ref: https://docs.phalcon.io/en/3.2/db-models-cache

Code from ref:

// Cache the resultset for only for 5 minutes
$products = Products::find(
    [
        'cache' => [
            'key'      => 'my-cache',
            'lifetime' => 300,
        ],
    ]
);
