Hi @stanleer you can get the count with this simple way


    $myCount = $this->getDi()->get('modelsManager')->createBuilder()
    ->columns(['myCount' => 'count(*)']) //myCount alias
    ->from(...)
    ->where(...)
    ->getQuery()
    ->setUniqueRow(true)
    ->execute()
    ->myCount; //myCount property
