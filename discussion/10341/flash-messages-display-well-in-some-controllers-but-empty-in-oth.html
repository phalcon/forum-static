I had some problems similar. I notice that the final view should have {% raw %}{{{% endraw %} content() {% raw %}}}{% endraw %} to output the previous rendered content which shows the messages coming from flash service. Ex:

in IndexController.php


class IndexController extends ControllerBase
{

    public function indexAction()
    {

    }

    public function homeAction(){

    }

}

In home.volt

{% raw %}{%{% endraw %} extends "layouts/base.volt" {% raw %}%}{% endraw %}

Hello World

{% raw %}{%{% endraw %} block container {% raw %}%}{% endraw %}
    {% raw %}{{{% endraw %} content() {% raw %}}}{% endraw %}
{% raw %}{%{% endraw %} endblock {% raw %}%}{% endraw %}

Hope this helps. Thanks.
