Hello all,

We have updated our main website to be multilingual. You will notice that there is a small dropdown at the top which allows you to choose your language.

The translation project is driven by Transifex (http://www.transifex.com) and the project page is located here:

https://www.transifex.com/projects/p/phalcon/

You will also notice that not all the available languages have everything translated and more so that there are languages missing. If you wish to help with the translation effort, feel free to request either the addition of a new language or to be part of the translation team of any given language.

The long term goal is to move the documentation in Transifex so that we can have a better and more multilingual site/docs.

Thanks to all!

If you have any issues, feel free to contact me at [email protected]
