I'm pretty positive that there is no support for oracle without PDO, even with PDO it's pretty scarce: https://github.com/phalcon/cphalcon/issues/12008

That issue is mainly about Phalcon 3, if you are using Phalcon 2 you can still use the Oracle adapter.

As for implementing your own DB adapter, this might prove useful: https://github.com/phalcon/incubator/blob/master/Library/Phalcon/Db/Adapter/Pdo/Oracle.php
