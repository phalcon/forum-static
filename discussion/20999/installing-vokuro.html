Did you modify your composer.json file? The default requirement for requirement for phalcon is ~4.0.0, which should allow 4.1.0. https://github.com/phalcon/vokuro/blob/master/composer.json

Next, make sure curl is installed. The errors you're getting are for the cli version of PHP, not the FPM version, so you'll need to modify a different php.ini file to make sure extensions are installed.
