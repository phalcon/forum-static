Bump!

For Apache users with the same issue, make sure you've got the rewrite module loaded,

a2enmod rewrite

..and that AllowOverride is set to "All" in the apache conf.

<Directory "/var/www/">
  ...
  AllowOverride All
</Directory>

Don't forget to restart apache.
