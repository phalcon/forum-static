namespace Igor;

abstract class MyClassA
{
   // some methods here
}
namespace Igor;

interface MyInterfaceB
{
 // some methods here
}
namespace Igor;

abstract class MyClassC extends Igor\MyClassA implements Igor\MyInterfaceB
{
 // methods here
}

p.s. if in this class not use "abstract" then all works fine.

error:

-----------------------------------------------------^

Have ideas?
