Is there a usual way to assign variables specially for the template using "include"?

{% raw %}{%{% endraw %} include "path/to/template.volt" {% raw %}%}{% endraw %}

I think if you are using {% raw %}{%{% endraw %} include {% raw %}%}{% endraw %} most of the time you include a partial template (let's say a variable x is included) into the main template and then $this->view->setVar("x", what you define as x ) in your main controller.

https://docs.phalcon.io/en/latest/reference/volt.html#view-integration

{# Passing extra variables #}
<div id="footer">{% raw %}{%{% endraw %} include "partials/footer" with ['links': links] {% raw %}%}{% endraw %}</div>
