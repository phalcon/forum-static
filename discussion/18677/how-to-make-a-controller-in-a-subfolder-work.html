SOLVED

Loader file Change from this

$loader->registerNamespaces(
    [
       'App\Controllers\Wsget' => 'app/controllers/wsget',
    ]
);

To this :

$loader->registerNamespaces(
    [
       'App\Controllers\Wsget' => __DIR__ . '/../controllers/wsget',
    ]
);
