In addition to Wojciech answer, if you need more complex queries you should always bind parameters like Xaero suggested.

Models:

$robots = Robots::find(
    [
        'name = :name: AND type = :type:',
        'bind' => [
            'name' => 'Robotina',
            'type' => 'maid',
        ],
    ]
);

Query Builder:

->where('name = :name:', ['name' => $name])
