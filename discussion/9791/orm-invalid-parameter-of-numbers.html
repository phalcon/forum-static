                    ->where("parent_id = :parent_id:")

Instead of:

->where("parent_id = :parent_id:")
->where("position >= :newIndex:")
->where("position < :oldIndex:")

use

->where("parent_id = :parent_id:")
->andWhere("position >= :newIndex:")
->andWhere("position < :oldIndex:")
