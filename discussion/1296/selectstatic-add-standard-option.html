I think you want just select() and not selectStatic(). You should be able to add something like:

{% raw %}{{{% endraw %} select(["g", genres, "using":["id", "name"], "class": "form-control", 'useEmpty': true, 'emptyText': 'Not chosen', 'emptyValue': '-1']) {% raw %}}}{% endraw %}
