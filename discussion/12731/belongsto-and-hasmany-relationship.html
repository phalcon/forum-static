I think "paper" would be a collection or an array. Try to loop or access within index:

{% raw %}{{{% endraw %} user.paper[0].diverse {% raw %}}}{% endraw %}

or

{% raw %}{%{% endraw %} for p in user.paper {% raw %}%}{% endraw %}
    {% raw %}{{{% endraw %} p.diverse {% raw %}}}{% endraw %}
{% raw %}{%{% endraw %} endfor {% raw %}%}{% endraw %}
