  • variant @Jurigag: set root dir
  • Vagrant Box use apache, If I remember it right, so just add .htaccess in /var/www . More read in https://docs.phalcon.io/en/latest/reference/apache.html
# /var/www/.htaccess

<IfModule mod_rewrite.c>
    RewriteEngine on
    RewriteRule  ^$ project-dir/    [L]
    RewriteRule  ((?s).*) project-dir/$1 [L]
</IfModule>
