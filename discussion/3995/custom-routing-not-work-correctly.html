I defined a route:

$di['router']->add('#^/([a-zA-Z0-9\_\-]+)/([0-9]+)#', array(

'params' is expected to being used alongside regular expressions that return an undefined list of parameters. You will not have this problem if you change it to:

$di['router']->add('#^/([a-zA-Z0-9\_\-]+)/([0-9]+)#', array(
    'controller' => 1,
    'action' => 'index',
    'parameters' => 2
));
