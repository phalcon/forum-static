Last year, part of developers moved their local web-server to virtual web-servers by Vagrant technology.

Today, I am going to show community my Vagrant machine.

This machine is used to develop applications on PHP with Phalcon 2/Zephir/Lynx.

https://github.com/ovr/perfect-php-vagrant

Software:

  • PHP 5.6
  • Mysql
  • Nginx
  • Zephir
  • Phalcon
  • Lynx
  • NodeJS/NPM/Bower/Gulp/Grunt

Pre-installed projects:

  • Localhost на http://vm.local/
  • AngularJS Skeleton на http://angular-skeleton.local/
  • Phalcon Module Skeleton на http://phalcon-module.local/

Star project, follow me if you are interested in my development :)

P.S If you didn't try to work with Vagrant, it's time to try ;)
