The phalcon docs say it partially implements v1.2 but the latest beanstalkd is 1.9. I would just try 1.9 and see if you run into any issues. You just might not have access to the newer beanstalk features if you use the Phalcon library. It doesn't look like it implements everything but I haven't set up beanstalk/phalcon yet.
