It's simple like this:

Install phalcon, and in CI just make use use of \Phalcon\Mvc\Model. So, instead of :

class Blogmodel extends CI_Model 
{
    var $title   = '';
    var $content = '';
    var $date    = '';
}

use 

class Blogmodel extends \Phalcon\Mvc\Model 
{
    var $title   = '';
    var $content = '';
    var $date    = '';
}
