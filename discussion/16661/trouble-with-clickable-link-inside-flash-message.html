Try like this in your service file:

        // Flash messages
        $di->setShared('flash', function() {
            $flash = new \Phalcon\Flash\Session();
            $flash->setAutomaticHtml(false);
            $flash->setAutoescape(false); // This is important part
            return $flash;
        });
