As you say, it returns an array ( https://github.com/phalcon/cphalcon/blob/2.0.0/phalcon/http/request.zep#L759 ): the problem is that is an array of Files ( http://docs.phalcon.io/es/latest/api/Phalcon_Http_Request_File.html ). So, unless it is changed, you should create your own function that searchs for the specific file.
