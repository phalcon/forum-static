Hi,

When reaching the end of the first section of the tutorial my browser shows a directory listing instead of the "Hello!" message using the following URL - "http://localhost/tutorial".

this looks like you haven't mod_rewrite enabled (and your .htacces file is ignored). Solution is to type command: a2enmod rewrite (as root)

1) What does the "test" refer to in the URL "http://localhost/test" as shown in the graphic saying "Hello!"?

it's an error, it should be (on the picture): "http://localhost/tutorial".

2) What is the root folder in Apache's httpd.conf set to in relation to the tutorial folder as shown in the suggested file structure?

looks like default apache config: usualy /var/www (under linux) => http://localhost /var/www/tutorial => http://localhost/tutorial
