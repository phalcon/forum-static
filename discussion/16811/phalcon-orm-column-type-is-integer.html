Hi @l770277 try with this

Table::find([
    'conditions' => '1 <> ALL({mix:array-int})', 
    'bind' => [
        'mix' => [1,2,3, ... ] // your integer[]
    ]
]);

{param:array-int} works with integer array. If you want string only use {param:array}

Then you can read binding parameters docs

Good luck
