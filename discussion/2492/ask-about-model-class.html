Hi, This code should be works:

$results = Coordenada::find(array(
    'columns' => 'latitude, longitude, MAX(data)'
    'group' => 'rota_id'
));

You can find information about this on models page: http://docs.phalcon.io/en/latest/reference/models.html#finding-records There is a table with list of available parameters which can be used in Model::find.
