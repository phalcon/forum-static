Are you talking about IN clause?

$robots = Robots::find([
    'id IN ({ids:array})',
    'bind' => [
        'ids' => [1, 2, 3, 4, 5]
    ]
]);

More info here: https://docs.phalcon.io/en/latest/reference/models.html#binding-parameters scroll little further after first two code sections;
