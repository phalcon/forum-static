When you use softdelete - you not delete rows in reallity. So you can't use :beforeDelete and :afterDelete. If you want find all "softdeleted" rows - you can do it:

Model::find("is_deleted=1");

or

Model::find(
    [
        'conditions' => 'is_deleted = ?1',
        'bind'       => [
            1 => 1,
        ]
    ]
);
