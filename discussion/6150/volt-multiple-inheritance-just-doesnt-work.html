  1. You cannot add {% raw %}{%{% endraw %} block javascript {% raw %}%}{% endraw %} in a child template, because it doesn't existe in the primary parent template (said in other way, you cannot add new content blocks in child templates).
  2. You have to use {% raw %}{{{% endraw %} super() {% raw %}}}{% endraw %} when retrieving content from the parent template.
  3. When you extend, you put the content into the child template, not the parent template. I think you should use include for the opossite effect (https://docs.phalcon.io/es/latest/reference/volt.html#view-integration). The above rules applies the same way for this (except for the second one).
