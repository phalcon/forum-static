You can create routes where a parameter is the extension:

$router->add(
    "/documentation/{chapter}/{name}\.{type:[a-z]+}",
    array(
        "controller" => "documentation",
        "action"     => "show"
    )
);

https://docs.phalcon.io/en/latest/reference/routing.html#parameters-with-names
