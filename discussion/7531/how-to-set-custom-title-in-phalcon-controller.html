How exactly is not working? Title is only to define a title in the browser toolbar. It won't show on your view page.

http://www.w3schools.com/tags/tag_title.asp

If you want to print the title on the view page put {% raw %}{{{% endraw %} get_title(false) {% raw %}}}{% endraw %}. This will disable the title tags and will be visible. But in that case I suggest you to set a different variable in controller to handle only the view title. The page title can be appended or prepended and later you might want to add the website name on it and you won't want to be shown on view.
