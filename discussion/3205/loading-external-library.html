According to the library documentation, you can use:

1) A Phar file:

<?php
// Point to where you downloaded the phar
include './httpful.phar';

// And you're ready to go!
$response = \Httpful\Request::get('https://example.com')->send();
...

or

2) Composer:

{
    "require": {
        "nategood/httpful": "*"
    }
}

Check how dependencies are loaded using Composer in Vokuro:

  • https://github.com/phalcon/vokuro/blob/master/composer.json#L5
  • https://github.com/phalcon/vokuro/blob/master/app/config/loader.php#L17
