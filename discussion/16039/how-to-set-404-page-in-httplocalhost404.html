I resolve my problem, just use this code:

$eventManager->attach('dispatch', function (Event $event, Dispatcher $dispatcher) {
                    if (!$dispatcher->getActionName()) {
                        $dispatcher->forward([
                            'controller' => 'error',
                            'action' => 'notFound',
                        ]);
                    }
                });
