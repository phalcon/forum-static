You mean this?

// Models
$all = News::find();

// QueryBuilder
$all = $this->modelsManager->createBuilder()
    ->from('Models\News')
    ->getQuery()->execute();

Both will return same results. Please note that if you use ->columns() on the QueryBuilder it will return simple object, not full News object.
