Yea, first argument is condition. Just write proper condition like "userId = :userId:", nothing fancy:

User::find([
    "userId = :userId:",
    "bind" => [
        "userId" => $allowedEntity->id
])

No, then you need to change it to:

$ids = [];
foreach($allowedEntity as $entity) {
    $ids[] = $entity->id;
}
$result = User::find([
    "userId IN ({ids:array})",
   "bind" => [
       "ids" => $ids
    ]
]);
