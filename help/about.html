About Phosphorum

Phosphorum is an engine for building flexible, clear and fast forums. It is used by:

  • Phalcon Framework Forum
  • Zephir Language Forum

You can adapt it to your own needs or improve it if you want. If you want to improve this forum please submit a pull request repository.

Please write us if you have any feedback.
